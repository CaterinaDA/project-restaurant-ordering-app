# Jimmy's Diner Ordering App

This project is a simple restaurant ordering interface built using HTML, CSS, and JavaScript. The user can add and remove items from their order, view the total price, enter payment details, and receive a confirmation message.

## Features

- Dynamic menu rendering using JavaScript modules
- Add and remove items from the cart
- Display quantity and total price calculations
- Remove individual items from the order
- Toggle visibility of the order section based on the cart state
- Modal form for payment details
- Thank-you message after completing the order

## Technologies Used

- HTML5 for layout and structure
- CSS3 for styling
- Vanilla JavaScript (ES modules) for functionality

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
3. Add items to the cart and proceed with checkout

No backend or database is required. All interactions run client-side.
