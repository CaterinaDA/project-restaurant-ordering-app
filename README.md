# Jimmy's Diner Ordering App

This project is a restaurant ordering interface built using HTML, CSS, and JavaScript. Users can add and remove menu items, review their order, enter payment details, and receive a confirmation message.

## Features

- Dynamic menu rendering using JavaScript modules
- Add and remove items from the cart
- Quantity and total price calculation
- Remove individual items from the order
- Order section visibility toggling based on cart state
- Modal form for payment details
- Thank-you confirmation message upon purchase

## Technologies Used

- HTML5 for structure
- CSS3 for styling
- Vanilla JavaScript (ES modules) for functionality

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
3. Add items to the cart and complete the checkout

No backend or database is required, as the application runs entirely on the client side.
